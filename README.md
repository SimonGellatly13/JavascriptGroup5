Observations from testing HTML: Phase 2

//categories.html
This page contains bakery item categories along with simple pictures representing each category. The search bars, filter options, and category buttons are not yet responsive.

//homepage.html
This page provides a short description regarding the contents of the webpage, along with popular recipes. The navigation bar is responsive. The search bar and links to popular recipes are not yet responsive.

//recipes.html
This page displays all the recipes the website contains. The navigation bar and recipe buttons are responsive. Search bar is not yet responsive. Each recipe card displays the name of the bakery item, a short description, serving sizes, measurement conversions, ingredients, instructions, and nutrition facts. The serving size dropdown button and the conversion button is not yet responsive.
