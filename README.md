<strong>The Desserts Club is a web application designed to help users explore and discover dessert recipes. The web application provides an organized, user-friendly experience with features that make following recipes easy interactive and fun.</strong>

<strong> Main Features </strong>
1. Homepage
- Overview of the website 
- Featured popular recipes section
- Navigation to recipes, categories, and favourites
- Search bar for quick recipe lookup
- Dark mode toggle


2. Recipes Page
- Lists all available recipes
- Search bar integrated
- Navigation back to homepage or categories
- Dark mode toggle 


3. Recipe Detail Pages
- <strong> Full recipe includes: </strong>
- Ingredients with a clickable checklist to track progress
- Step-by-step instructions
- Nutrition facts
- Scaled servings functionality
- Dark mode toggle
- Search bar for navigating to other recipes


4. Categories Pages
- Browse recipes by category Cakes, Tarts, Cookies, Mousses, Cheesecakes
- Recipe cards for each item with images and links
- Search bar 
- Dark mode toggle


5. Favourites Page
- Users can view saved favourite recipes
- Grid layout for easy navigation
- Dark mode toggle
- Search bar for filtering favourites


6. Shared Features Across All Pages
- Search functionality fetching recipes from JSON
- Dark mode toggle that is persistent 
- Responsive design for mobile and desktop
- Consistent navigation bar linking all pages
